# Git-Lite (Java)

This project is a **Java-based Git-Lite** that mimics essential Git/GitHub operations such as:

- Creating a GitHub profile  
- Creating repositories  
- Adding files  
- Staging & committing changes  
- Viewing commit history  
- Searching repositories using **custom-built hashing** (without Java built-in DS)

This system is built **from scratch in Java**, using:
- Custom Hash Table  
- Custom Linked Lists  
- No built-in Java data structures (No HashMap, No ArrayList)

---

## 🚀 Features

### **1. User Profile Management**
- Create GitHub profile  
- Store username, email, bio  
- Custom hashing to store user in Hash Table  

### **2. Repository Management**
- Create unlimited repositories  
- Store repository details using custom linked list  
- Search repositories by name using hash function

### **3. File Management**
- Add files to a repository  
- Each file is stored using custom node structure  
- File history saved manually

### **4. Git-Like Commit System**
- Stage file changes  
- Commit to repository  
- Commit history saved with:
  - Commit ID  
  - Timestamp  
  - Message  
  - Updated file list  

### **5. Search Functionality**
- Search repo using hashing  
- Search commits using linear search  
- Search files inside the repository
